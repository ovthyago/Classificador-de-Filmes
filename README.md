# Classificador-de-Filmes

Esse projeto consiste na desenvolvimento de documentação de uma aplicação que classifica filmes, existentes em uma plataforma de streaming, quanto aos seus gêneros. O desenvolvimento desse projeto se torna necessário devido a plataformas não
realizarem de forma correta a classificação, associando termos que não existem, a exemplo de termos como "bizarro" ou "mistério", no lugar de associar a gênero notórios, como "suspense", "terror", "Drama" e dentre outros.
